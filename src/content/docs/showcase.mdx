## Code Block Example

```bash title="Deploy with Docker"
# Pull the latest image
docker pull your-app:latest

# Run with resource limits
docker run -d \
  --name your-app \
  --memory=4g \
  --restart unless-stopped \
  -p 80:3000 \
  your-app:latest
```
