## Quick Decisions

### For a new SaaS project:
- **Hosting:** Vercel (best DX)
- **Auth:** Clerk (fastest setup)
- **Database:** Turso or Supabase

### For a privacy-focused project:
- **Hosting:** Cloudflare (edge, privacy)
- **Auth:** BetterAuth (self-hosted)
- **Database:** Your own PostgreSQL

### For a content site:
- **Hosting:** Cloudflare Pages (free, fast)
- **CMS:** Sanity (structured content)
- **Framework:** Astro (static-first)

### For a high-traffic app:
- **Hosting:** Vercel + Cloudflare (hybrid)
- **Auth:** Clerk (managed, scalable)
- **Database:** PlanetScale or Neon
