## Quick Install Commands

Copy these to get started with each platform:

### Vercel CLI
```bash
npm i -g vercel
```

### AI SDK
```bash
npm i ai
```

### BetterAuth
```bash
npm i better-auth
npx @better-auth/cli init
```

### Clerk
```bash
npx clerk@latest
```

### Sanity
```bash
npm create sanity@latest
```

## Using with Claude Code

Ask Claude to run any of these commands for you:

> "Deploy my project to Vercel production"

Claude will run `vercel deploy --prod` and show you the deployment URL.

> "Pull my environment variables from Vercel"

Claude will run `vercel env pull .env.local` to sync your local env file.

> "Set up BetterAuth in this project"

Claude will run the interactive setup wizard and guide you through configuration.
