A curated collection of free resources to significantly improve the UI and front-end design of your applications. These tools fill the gap where AI coding tools often fall short on visual design.

## Component Libraries

Ready-to-use UI components that can be copied directly into your projects.

### 21st.dev

**Best for:** Ready-to-use component code with AI coding tools

A UI component library with hundreds of ready-to-use templates. Copy the component code as a prompt and paste it directly into AI coding tools like Lovable or Cursor for instant implementation.

**Key Features:**
- Hero sections, navigation, forms, and more
- Copy code as prompts for AI tools
- Easy to modify components to match brand styles

[Visit 21st.dev](https://21st.dev)

---

### UIVerse.io

**Best for:** Open-source UI elements with multiple export formats

A large library of open-source UI elements built by the community. Export components in multiple formats including HTML, CSS, Tailwind, React, or Figma.

**Key Features:**
- Community-built components
- Filter by type (forms, cards, buttons, etc.)
- Multiple export formats (HTML, CSS, Tailwind, React, Figma)

[Visit UIVerse.io](https://uiverse.io)

## Design Inspiration

Platforms for finding UI design inspiration from professional designers and real-world websites.

### Dribbble.com

**Best for:** High-quality design inspiration from top designers

An "Instagram for designers" where top designers share their work. Ideal for finding UI inspiration for dashboards, landing pages, or other interfaces.

**How to Use:**
1. Search for keywords (e.g., "dashboard design", "SaaS landing page")
2. Download images of designs you like
3. Use AI tools to draw inspiration from layout, color palette, or typography

[Visit Dribbble.com](https://dribbble.com)

---

### Landingfolio.com

**Best for:** Real landing page designs from actual companies

A curated portfolio of real landing page designs used by actual companies. See designs in action by visiting the live websites.

**Key Features:**
- Real-world landing page examples
- Filter by page type (pricing, features, etc.)
- Links to live websites

[Visit Landingfolio.com](https://landingfolio.com)

---

### Navbar.gallery

**Best for:** Navigation bar design inspiration

Specifically dedicated to navigation bars, menus, and sidebars. Curates well-designed navbars from top websites.

**Key Features:**
- Focused exclusively on navigation elements
- Collection from top websites
- Sleek and functional navigation examples

[Visit Navbar.gallery](https://navbar.gallery)

---

### Land-book.com

**Best for:** Curated landing pages across niches with technical details

A curated collection of real landing pages across various niches and categories. Shows color palettes and technologies used.

**Key Features:**
- Recently added designs
- Color palette extraction
- Technologies used by each site
- Multiple categories and niches

[Visit Land-book.com](https://land-book.com)
